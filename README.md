# Научпоп компасс

Шуточная карта научно-популярных YouTube-каналов в духе политического компаса, но без политики.

Страница в этом репозитории: https://airshipster.github.io/scitopus-compass/

Оригинальная интерактивная версия: https://scitopus.com/compass

Оси сравнивают не качество каналов и не то, кто «научнее» в абсолютном смысле, а стиль подачи:

- сверху — строже и глубже;
- снизу — понятнее широкой аудитории;
- слева — более развлекательно;
- справа — более познавательно.

Это юмористическая визуализация, а не рейтинг, не экспертный вердикт и не попытка исключить кого-то из «настоящей науки». Раскладка может быть неточной: она строится автоматически по доступным данным и экспериментальным правилам.

![Научпоп компасс](assets/nauchpop-compass.jpg)

## Что внутри

На карте размещены научно-популярные YouTube-каналы из базы SciTopus. Точки показаны аватарками каналов, чтобы карту было проще считывать визуально.

В интерактивной версии можно:

- искать канал по названию;
- включать и выключать подписи;
- показывать только партнёров или все каналы из базы;
- фильтровать каналы по разделам;
- приближать и двигать карту;
- открывать карту на весь экран;
- кликать по каналу и смотреть его карточку;
- переходить на YouTube-канал.

## Источники расчёта

Координаты можно пересчитывать прямо на странице, включая и отключая разные источники:

- теги канала;
- описание канала;
- названия последних 50 видео;
- теги последних 50 видео;
- категория базы SciTopus;
- название канала;
- эвристика.

Чем меньше источников включено, тем выше шанс, что часть каналов окажется ближе к центру или в похожих позициях. Это не ошибка сама по себе: если данных мало или выбранный признак почти одинаковый для группы каналов, карте нечем уверенно развести точки.

## Эвристика

«Эвристика» - отдельный режим старой раскладки. Он использует заранее заданные правила, слова-маркеры, категории и теги канала. Это не ручная расстановка автором SciTopus и не персональный вердикт по каждому каналу.

Единственная точка, закреплённая вручную, - SciTopus в центре карты.

## Режим «возвести в абсолют»

Этот режим растягивает текущую раскладку к краям компаса, чтобы различия между каналами было легче увидеть. Он не меняет смысл осей, а только усиливает расстояния между уже рассчитанными координатами.

## Ограничения

Компас сделан как шутка и визуальный эксперимент. Он может ошибаться, особенно если у канала мало доступных тегов, описаний или свежих видео. Категории базы и названия каналов могут быть субъективными или шумными признаками, поэтому их можно отключить.

Если канал оказался «не там», это не обвинение и не оценка качества. Это повод поспорить с алгоритмом, а не с каналом.

## Технически

Страница лежит в репозитории как один автономный `index.html`: внутри находятся HTML, CSS, JavaScript и данные для карты. Аватарки каналов подгружаются по внешним URL.

Скриншот для описания лежит в `assets/nauchpop-compass.jpg`.
